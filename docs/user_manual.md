Welcome to Proof of Play! This manual will help you get started and make the most of your experience.

## Getting Started
1. Make sure you have Python 3.8 or higher installed.
2. Open a terminal and navigate to the project folder.
3. Run the game:
   ```sh
   python -m ProofPlayGame.main
   ```

## How to Play
- Enter your name when prompted.
- Each level presents a logic or programming question.
- Select your answer by typing A, B, C, or D and pressing Enter.
- If you answer incorrectly three times, a hint will appear.
- Too many wrong attempts will reset the level (no penalty).
- Some answers will branch you to different levels or paths, making each playthrough unique.
- Complete all levels to win the game!

## Game Features
- **Branching Paths:** Your choices affect your journey.
- **Randomized Questions:** Each session is different.
- **Hints:** Get help after 3 wrong answers.
- **No Penalty Resets:** Levels reset after repeated mistakes, but you can always try again.

## Troubleshooting
- If the game does not start, ensure you are running Python 3.8+.
- If you see an error, check that all files are present and you are in the correct directory.
- For any issues, refer to the README or contact the project maintainer.

Enjoy the challenge and have fun learning!
